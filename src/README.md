# Source Code Directory

This directory contains the main source code for Vibeforge.

## Structure

```
src/
├── music/          # Music engine and beat generation
├── stories/        # Story writing and screenplay tools
├── ai-math/        # AI-powered mathematics
├── marketplace/    # Digital marketplace
├── auth/           # Authentication system
├── ui/             # User interface components
└── utils/          # Utility functions
```

Each subdirectory contains feature-specific code.
