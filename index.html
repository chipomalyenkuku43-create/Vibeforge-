How to run locally

  1. Clone the repo and checkout the branch: git clone https://github.com/chipomalyenkuku43-create/Vibeforge-.git && cd Vibeforge- && git checkout fix/app-setup
  2. Start a simple static file server (recommended):
    • Python 3: python -m http.server 8000
    • Node (serve): npx serve .
  3. Open http://localhost:8000 in your browser.

If you want this hosted on GitHub Pages, enable Pages in the repo settings and set the branch to fix/app-setup (or merge to main and use main branch for Pages).
